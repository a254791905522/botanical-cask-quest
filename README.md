# Botanical Cask Quest Release Package

App name: Botanical Cask Quest

Bundle ID: com.rosewood.botanicalcaskquest.game

Device support: iPhone only

Gameplay fact card:
- Core action: Operate a 1890s copper still, select materials and process steps, cut heads hearts and tails, age in oak casks, and bottle craft spirits.
- Main puzzle: Match the correct raw material and process flow to the spirit type, then heat, cut, age, and bottle within the shift limit.
- Instrument system: Boiler temperature (0-120C), condenser temperature (5-25C), proof tracking, heads/hearts/tails volume in mL.
- Still types: Pot Still, Column Still, Continuous Still, Vacuum Still.
- Cask types: Bourbon, Sherry, New Oak, Wine, Rum.
- Progression: 50 contracts across 5 historic distillery regions (London Gin, Ireland Whiskey, Cognac Brandy, Poland Vodka, Barbados Rum).
- Monetization and network: offline, no ads, no in-app purchases, no account, no analytics, no tracking.
